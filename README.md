# Mentoso
Mentoso : Mental health detection and Counselling application 
Spearheaded the development of MENTOSO, a comprehensive mental health assessment and counseling platform. 
 The project aimed to enhance patient-psychiatrist connectivity through an intuitive and user-friendly interface.
 Technology Stack : HTML5, CSS3, JavaScript, MongoDB, AWS, Git, VsCode, PyCharm, Postman, Jira, Machine Learning (NLP).
